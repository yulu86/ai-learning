# Music Store Demo

[`jupyter notebook` 使用](./README.md) | 音乐商店Demo

## 激活`conda`环境

```bash
conda activate ml
```

> 注意：Mac环境以下命令中 `python` 替换为 `python3`

## Jupyter Notebook

### 启动 `jupyter notebook` 服务

```bash
jupyter notebook
```

### 访问 `jupyter notebook`

- 访问 [http://localhost:8888/tree](http://localhost:8888/tree)

### 下载数据集

- 访问 [music csv](http://bit.ly/music-csv)
