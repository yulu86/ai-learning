## 环境准备

- 创建虚拟环境

```bash
conda create -n ml-math python=3.10.6
conda activate ml-math
```

> 注意：Mac环境以下命令中 `python` 替换为 `python3`

- 安装依赖

```bash
python -m pip install -r requirements.txt -i https://pypi.tuna.tsinghua.edu.cn/simple
```

- 启动`jupyter notebook`

```bash
jupyter notebook
```
